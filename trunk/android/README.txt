
Development Environment Installation Instructions

1. Install Java JDK from http://www.oracle.com/technetwork/java/javase/downloads/index.html

2. Download and install (extract) Eclipse Java EE (note that Eclipse Classic will NOT work with the Android Development Tools (ADT) plugin)

3. Install Android Development Tools and SDK from

* http://developer.android.com/sdk/index.html

4. From the Android SDK Manager, install:

* Tools -> Android SDK Tools and Android SDK Platform-tools
* Android 2.3.3 (API10) -> SDK Platform, Google APIs, and Samples for SDK
* Extras -> Google USB Driver

5. Install Eclipse ADT Plugin: http://developer.android.com/sdk/eclipse-adt.html

* In Eclipse, go to Help->Install New Software, enter the following URL, and select all: https://dl-ssl.google.com/android/eclipse/

* If https doesn't work, try http
